# NTU-EEAT
[109-1] Web Programming Final

### Usage
+ Download the code 
    ```bash
    git clone https://github.com/jamesyoung0623/NTU-EEAT.git
    ```
+ Install dependencies
    ```bash
    yarn
    ```
+ Run server
    ```bash
    yarn server
    ```

+ Run client
    ```bash
    yarn start
    ```
